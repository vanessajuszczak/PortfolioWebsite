# Portfolio Website

This is my personal portfolio website created for the Web UI Design module. The website was built using Sveltekit and shows off my work in games, digital art, animation, and 3D modelling. It includes the following pages:
- Homepage
- About page
- Projects page
- Project category pages
- Individual project pages
- CV page
- Contact page

The site was designed from my wireframes and developed to be responsive on both desktop and mobile devices.
